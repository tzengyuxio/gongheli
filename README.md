# 共和曆 (Gonghe Calendar)

> 聖人慎守日月之數，以察星辰之行，以序四時之順逆，謂之厤 ——《大戴禮記·曾子天圓》

共和曆是一套以西周共和元年為紀元的曆法系統。

**簡以致用，精以合天**

## 核心特點

### 紀元

- 以**西周共和元年**為元年（共和 0 年），這是人類連續紀年的開始
- 有**零年**的設計，消除了現行公元紀年的零年問題

### 歲首

- 以**冬至**為歲首（1 月 1 日）

### 月制

- 一年 12 個月
- 單數月（1, 3, 5, 7, 9, 11）：**30 天**
- 偶數月（2, 4, 6, 8, 10）：**31 天**
- 12 月：平年 **30 天**，閏年 **31 天**
- 平年 365 天，閏年 366 天

### 閏法

- 四年一閏，逢 **128 的倍數不閏**
- 此規則下太陽日與曆日的誤差為所有簡單閏法中最小

### 紀 (Era)

- 定 **4418 年**為一紀
- 4418 年 = 54,643 月 = 1,613,640 日 = 26,894 甲子 = 230,520 週
- 此為回歸年、朔望月、太陽日及一甲子的最小公倍數
- 日數同時是干支紀日與七曜星期的整數週期倍數
- 年、月、日的倍數，象徵日（太陽年）、月（朔望月）與人（干支紀日）三者的和諧

### 子輿日 (Ziyu Day Number)

- 類似儒略日的連續紀日系統，紀念**曾子**（名參，字子輿）
- 以午夜為日的起始（非正午）
- 第 0 紀第一天為子輿日起點
- **共和 0 年 1 月 1 日 = 子輿日 800,609**
- 子輿日首日相當於**儒略日 613,270.5**
- 涵蓋人類有文字、文明以來的歷史階段，可描述任何考古紀年

### 紀元起算

- **第一紀第一天**：西元 1384 年 12 月 13 日／明洪武十七年（朔旦冬至甲子日）
- 往前推 4418 年（1,613,640 日）為第 0 紀第一天（子輿日起點）

## 相較現行曆法的優勢

- 消除格里曆的零年問題
- 日期與天象對齊，無偏離混亂
- 不存在各國採用格里曆時間不同所造成的計日混亂
- 閏法規則簡潔且精確度更高

## 設計理念

共和曆的目標是一套**跨越空間與時間的通用曆法**——不偏向任何單一文化或歷史起點，而是為所有人建立一套公平、精確的時間座標系。這個理念與世界語（Esperanto）的精神相近：以人工設計的方式消弭隔閡，讓不同背景的人都能平等地使用同一套語言或系統。本專案的前身 `kalendaro`，正是取自世界語中「曆法」一詞。
